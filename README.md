## MOVING 영화 검색, 리뷰 사이트 입니다.
* 궁금한 영화가 있을 때 영화 정보 사이트에서 평가를 자주 봤던 기억이 나서 기획하게 되었습니다.
* 사이트에 등록된 영화들의 정보를 조회할 수 있고, 누구나 리뷰를 남길 수 있어 주관적인 평가들을 확인할 수 있습니다.
